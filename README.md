# 🏨 Hotel Management System

A full-stack web application for managing hotel operations built with Laravel, HTML, CSS, and MySQL.

## 🌟 Features
- User Registration & Login
- Room Management (Add, View, Update, Delete)
- Booking Management
- Contact Page
- Admin Dashboard
- MySQL Database via XAMPP

## 🛠️ Built With
- Laravel (PHP Framework)
- HTML & CSS
- MySQL (XAMPP)

## ⚙️ How to Run Locally
1. Clone the repo
2. Install XAMPP and start Apache & MySQL
3. Import the database from the `database` folder
4. Run `composer install`
5. Copy `.env.example` to `.env` and set your database name
6. Run `php artisan serve`
7. Open `http://localhost:8000`
